\doxysubsection{Detailed Description}
Class representing a condition in the WHERE clause of a SQL query. 

Has three member variables, field, op, and value which describe a condition as\+: \{field\} \{op\} \{value\} For example\+: Item\+ID = 123 Price \texorpdfstring{$>$}{>}= 2.\+00
